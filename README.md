## 项目简介

本项目是一个基于 Python 的教务系统 Web 自动化流程学习示例，主要用于练习浏览器自动化、表单填写、验证码识别、页面状态判断和异常处理等技术点。
项目仅用于 Python 自动化学习与技术研究，不建议在真实教务系统中进行高频请求或用于影响正常选课秩序的行为。


## 核心功能

- 教务系统登录流程自动化示例
- 表单输入与页面元素定位
- OCR 验证码识别流程测试
- 页面状态判断与异常处理
- 自动化流程日志输出

- 核心语言：Python 3
