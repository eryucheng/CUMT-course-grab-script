# CUMT-course-grab-script
由于作者选课的时候没抢到课，为了防止学分不够写的补选课抢课工具,最后也是以此成功抢到课了
![自动抢课运行效果](战果.png)

## 项目介绍
针对校内教务系统抢课效率低、验证码手动输入繁琐的痛点，开发的自动化工具，实现**登录、验证码识别、课程搜索、报名**全流程自动化，提升选课效率。

## 核心功能
- 教务系统自动登录，支持账号密码安全输入
- 集成 ddddocr 自动识别验证码，无需手动操作
- 自定义课程号、报名原因，精准抢课
- 异常捕获与流程提示，运行稳定

## 技术栈
- 核心语言：Python 3
- 自动化框架：DrissionPage
- 验证码识别：ddddocr
- 工具：网页元素定位、异常处理、流程自动化

## 快速开始
### 1. 安装依赖
```bash
pip install -r requirements.txt
```
2. 运行脚本
```bash
python course_grab.py
```
4. 按提示输入教务系统学号 + 密码，自动执行抢课
